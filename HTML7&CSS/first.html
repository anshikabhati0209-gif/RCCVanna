HTML & CSS ASSIGNMENT I

THE BLOCK LEVEL ELEMENTS ARE:

Block level elements generally start on a new line and take up the full width of their parent.

  • Div Tag(div): Used to group-content together.
  • Paragraph Tag(p): Represent a paragraph of text.
  • Heading Tag(H1-H6): Headings from the largest(H1) to smallest(H6).
  • Unordered List Tag(ul): Represents an Unordered List.
  • Ordered List Tag(ol): Represents an Ordered List.
  • List Tag(li): Represents a list item, inside (ul)and(ol) tags.
  • Header Tag(header): Represents the header section of a page or section.
  • Footer Tag(footer): Represents the footer section of a page or section
  • Section Tag(section): Represents the section of a content
  • Article Tag(article): Represents independent content, like a news, article or blog post

THE INLINE ELEMENT ARE:

Inline Element do not start on a new line and take up only as much width as necessary.

  • Span Tag(span): Used to group inline content, like text.
  • Anchor Tag(a): Represents a hyperlinks.
  • Strong Tag(strong): Represents a important text(usually bold).
  • Emphasize Tag(em): Represents emphasized text(usually italicized).
  • Image Tag(img): Represents a image.
  • BreakLine Tag(br): Represents a break in the line.
  • Italic Style Tag(i): Represents text in an italic style.
  • Bold Tag(b): Represents text in a bold style.
  • Subscript Tag(sub): Represents subscript text, typically used for chemical formulas or mathematical notation.
  • Superscript Tag(sup): Represent superscript text, typically used for exponents or footnotes.
